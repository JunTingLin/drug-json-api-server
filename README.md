# drug-json-api-server


https://my-json-server.typicode.com/JunTingLin/drug-json-api-server/drugs

| 目的       | 方法        | URL         |
| -------- | --------- | ----------- |
| 取得所有藥物資料 | GET       | /drugs      |
| 取得某藥物資料  | GET       | /drugs/1    |
| 新增藥物     | POST      | /drugs      |
| 更改某藥物資料  | PUT/PATCH | /drugs/{id} |
| 刪除某藥物    | DELETE    | /drugs/{id} |
